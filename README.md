# ⚽ Smart Draft Board - Soccer Player Draft Application

A web-based interactive draft board for soccer players, built with Flask and Tailwind CSS.

## Features

- 📤 Import players from CSV/Excel files (compatible with Google Sheets and FBRef exports)
- 🔄 Smart column mapping for player name, position, and club
- 🎯 Real-time drafting with undo/redo functionality
- 🔍 Search and filter players by name, club, or position
- 🎲 Random pick generator
- 📱 Fully responsive design
